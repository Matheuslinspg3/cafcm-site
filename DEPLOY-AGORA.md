# 🚀 Deploy IMEDIATO no Cloudflare Pages

## ✅ PRONTO PARA USAR!

Esta versão usa **mailto** - não precisa de API Key, Web3Forms ou nada externo!

---

## 📦 Como Funciona

**Formulário de Contato:**
1. Usuário preenche o formulário
2. Clica em "Enviar Mensagem"
3. Abre o Gmail/Outlook dele
4. Email já vem preenchido com os dados
5. Ele só precisa clicar em "Enviar"

**Vantagens:**
✅ Funciona 100% sem backend
✅ Sem limite de mensagens
✅ Sem cadastros em serviços externos
✅ Deploy imediato

**Limitação:**
⚠️ Usuário precisa ter email configurado no computador/celular

---

## 🌐 FAZER DEPLOY AGORA (2 minutos)

### **Passo 1: Acesse Cloudflare Pages**
```
https://pages.cloudflare.com/
```

### **Passo 2: Crie Conta (Grátis)**
- Use Google ou GitHub para login rápido

### **Passo 3: Upload**
1. Clique em "Create a project"
2. Clique em "Upload assets"
3. Arraste TODOS os arquivos desta pasta
4. Nome do projeto: `cafcm`
5. Clique em "Deploy site"

### **Passo 4: Aguarde 1 minuto**
⏳ Cloudflare está fazendo o deploy...

### **Passo 5: Pronto! 🎉**
```
https://cafcm.pages.dev
```

### **Opção 2: Formspree**
- 50 emails/mês grátis
- Aceita anexos de currículo
- https://formspree.io/

## 🎯 Domínio Customizado

Depois do deploy:

1. No painel do Cloudflare Pages
2. Clique em "Custom domains"
3. Adicione: `www.cafcm.org.br`
4. Siga instruções de DNS
